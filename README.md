# RIHA Legal Marketplace

Plugin marketplace for **RIHA legal advokátní kancelář s.r.o.** and **AWRA insolvence, v.o.s.**

## Plugins

| Plugin | Description |
|--------|-------------|
| **czech-legal-lookup** | Search Czech legislation |
| **insolvency-register** | Query ISIR insolvency proceedings |
| **datova-schranka-helper** | Format data box submissions |

## Usage

Add this marketplace in CODEXIS:
- **Source Type:** Git Repository
- **Repository URL:** `https://github.com/YOUR_USERNAME/riha-legal-marketplace.git`
- **Branch:** `main`
- **Manifest Path:** `.claude-plugin/marketplace.json`
